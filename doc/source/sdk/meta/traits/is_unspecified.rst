.. _meta_is_unspecified:

is_fundamental
==============

.. index::
    single: is_unspecified (meta)
    single: meta; is_unspecified

Description
^^^^^^^^^^^
Returns true if the type is of hierarchy :ref:`tag_unspecified_` and false otherwise.

^^^^^^^^^^^^^^^^^^^

  | Parameter | Requirement                  | Description                                              |

  | T         | None                         | Input type                                               |
  +-----------+------------------------------+----------------------------------------------------------+
