.. _meta_as_real:

as_real
==========

.. index::
    single: as_real (meta)
    single: meta; as_real

Description
^^^^^^^^^^^
Returns the input type rebound with the floating-point type with the same size as
its primitive type.

Template Parameters
^^^^^^^^^^^^^^^^^^^

  +-----------+--------------------------------------+------------------------------------------+
  | Parameter | Requirement                          | Description                              |
  +===========+======================================+==========================================+ 
  | T         | :ref:`concept_has_primitive_factory` | Input type                               |
  +-----------+--------------------------------------+------------------------------------------+

Model
^^^^^
|metafunction|_

Header File
^^^^^^^^^^^

  #include <nt2/sdk/functor/meta/as_real.hpp>

^^^^^^^^

.. code-block:: cpp

  {
    template<class T>
    struct as_real;
  }

Expression Semantics
^^^^^^^^^^^^^^^^^^^^

.. code-block:: cpp

  typedef nt2::meta::as_real<T>::type r;

**Semantic:** Equivalent to:

.. code-block:: cpp

  typedef make_real<sizeof(primitive_of<T>::type), factory_of<T>::type>::type r;
